# 硬件端 API 接口指南

> 本文档面向硬件端开发人员，列出核心的后端接口

## 基本信息

- **API基础地址**: `http://{server}/api/v1`
- **认证方式**: 无需用户认证，直接调用
- **请求格式**: JSON
- **响应格式**:
```json
{
    "code": 0,
    "message": "success",
    "data": { ... }
}
```

---

## 核心接口（必须实现）

### 1. 设备注册 `POST /device/register`

设备上电后必须先注册，才能被小程序绑定。

**请求**:

```json
{
    "device_sn": "BB20240001",
    "device_name": "宝宝婴儿床",
    "device_model": "BB-Pro-V1"
}
```

device_model 是设备型号/产品型号，如果暂时不需要区分型号，可以不传或传空值：

  {
      "device_sn": "BB20240001",
      "device_name": "宝宝婴儿床",
      "device_model": null
  }

**响应**:

```json
{
    "code": 0,
    "message": "设备注册成功",
    "data": {
        "id": 1,
        "device_sn": "BB20240001",
        "device_name": "宝宝婴儿床"
    }
}
```

---

### 2. 心跳 `POST /hardware/heartbeat`

**重要**: 这是获取 `baby_id` 的唯一方式！

硬件端定期发送心跳（每60秒），后端返回设备绑定状态。

**请求**:
```
POST /api/v1/hardware/heartbeat?device_sn=BB20240001&online_status=1
```

**响应**:
```json
{
    "code": 0,
    "message": "success",
    "data": {
        "device_sn": "BB20240001",
        "is_bound": true,
        "baby_id": 92,
        "family_id": 10,
        "work_mode": "sleep_bed",
        "online_status": 1,
        "last_bind_at": "2026-06-10T15:30:00"
    }
}
```

**响应字段**:

| 字段 | 类型 | 说明 |
|------|------|------|
| `is_bound` | boolean | 是否已绑定宝宝 |
| `baby_id` | int/null | 绑定的宝宝ID，未绑定时为 null |
| `family_id` | int/null | 家庭ID |
| `work_mode` | string | 当前工作模式 |
| `online_status` | int | 1=在线, 0=离线 |

---

### 3. 上传传感器数据 `POST /sensor/upload`

**请求**:
```json
{
    "device_sn": "BB20240001",
    "baby_id": 92,
    "collected_at": "2026-06-10T15:30:00",
    "breath_rate": 18.5,
    "heart_rate": 95.2,
    "body_movement": 0.3,
    "distance_cm": 45.2,
    "pose_status": "supine",
    "face_detected": 1,
    "expression": "calm",
    "room_temp": 25.5,
    "humidity": 55.0
}
```

**说明**:
- `baby_id` 可选，未传时后端自动通过 `device_sn` 查询
- `collected_at` 为必填，格式: `YYYY-MM-DDTHH:MM:SS`
- 所有字段都是可选的，按需上传即可

**响应**:
```json
{
    "code": 0,
    "message": "数据上传成功",
    "data": {
        "id": 12345
    }
}
```

---

### 4. 上传婴儿状态 `POST /sensor/status/upload`

**重要**: 由硬件端本地AI判断状态后上传！

**请求**:
```
POST /api/v1/sensor/status/upload?device_sn=BB20240001&status_type=sleeping&status_level=0&started_at=2026-06-10T15:30:00&breath_rate=18.5&heart_rate=95.2
```

**参数说明**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `device_sn` | string | 是 | 设备序列号 |
| `status_type` | string | 是 | 状态类型 |
| `status_level` | int | 是 | 风险等级 |
| `started_at` | datetime | 是 | 状态开始时间 |
| `baby_id` | int | 否 | 宝宝ID |
| `breath_rate` | float | 否 | 呼吸频率 |
| `heart_rate` | float | 否 | 心率 |
| `pose_status` | string | 否 | 姿态 |
| `expression` | string | 否 | 表情 |

**status_type 可选值**:

| 值 | 说明 |
|----|------|
| `sleeping` | 熟睡 |
| `awake` | 刚醒 |
| `playing` | 高兴玩耍 |
| `crying` | 哭闹 |
| `danger` | 危险动作 |

**status_level 可选值**:

| 值 | 说明 |
|----|------|
| 0 | 正常 |
| 1 | 轻微关注 |
| 2 | 警告 |
| 3 | 危险 |

**响应**:
```json
{
    "code": 0,
    "message": "状态上传成功",
    "data": {
        "id": 12345,
        "status_type": "sleeping",
        "status_level": 0
    }
}
```

## 完整流程

```
┌─────────────────────────────────────────────────┐
│ 1. POST /device/register                        │
│    注册设备                                     │
│ │                            │
│                    ▼                            │
│ 2. while True:                                 │
│    │ │
│    ├──► POST /hardware/heartbeat              │
│    │ 获取 baby_id                           │
│    │                                           │
│    ├──► POST /sensor/upload                   │
│    │    上传传感器原始数据                     │
│    │                                           │
│    ├──► POST /sensor/status/upload            │
│    │    上传AI判断的状态结果                   │
│    │                                           │
│    └──► sleep(60)                              │
└─────────────────────────────────────────────────┘
```

## 示例代码

```python
import requests
import time
from datetime import datetime

API_BASE = "http://api.example.com/api/v1"

class DeviceClient:
    def __init__(self, device_sn: str):
        self.device_sn = device_sn
        self.baby_id = None
        self.is_bound = False

    def register(self) -> bool:
        resp = requests.post(f"{API_BASE}/device/register", json={
            "device_sn": self.device_sn,
            "device_name": "宝宝婴儿床",
            "device_model": "BB-Pro-V1",
        })
        return resp.json().get("code") == 0

    def heartbeat(self) -> dict:
        resp = requests.post(f"{API_BASE}/hardware/heartbeat", params={
            "device_sn": self.device_sn,
            "online_status": 1,
        })
        data = resp.json()
        if data["code"] == 0:
            result = data["data"]
            self.is_bound = result["is_bound"]
            self.baby_id = result["baby_id"]
        return data

    def upload_sensor(self, timestamp: str, breath_rate: float, heart_rate: float) -> dict:
        payload = {
            "device_sn": self.device_sn,
            "baby_id": self.baby_id,
            "collected_at": timestamp,
            "breath_rate": breath_rate,
            "heart_rate": heart_rate,
        }
        resp = requests.post(f"{API_BASE}/sensor/upload", json=payload)
        return resp.json()

    def upload_status(self, status_type: str, status_level: int, breath_rate: float = None, heart_rate: float = None) -> dict:
        params = {
            "device_sn": self.device_sn,
            "status_type": status_type,
            "status_level": status_level,
            "started_at": datetime.now().isoformat(),
        }
        if breath_rate:
            params["breath_rate"] = breath_rate
        if heart_rate:
            params["heart_rate"] = heart_rate
        resp = requests.post(f"{API_BASE}/sensor/status/upload", params=params)
        return resp.json()


# 使用
device = DeviceClient("BB20240001")
device.register()

while True:
    device.heartbeat()
    print(f"绑定: {device.is_bound}, baby_id: {device.baby_id}")

    # 上传传感器数据
    device.upload_sensor(
        timestamp="2026-06-10T15:30:00",
        breath_rate=18.5,
        heart_rate=95.2,
    )

    # 上传状态（本地AI判断结果）
    device.upload_status(
        status_type="sleeping",
        status_level=0,
        breath_rate=18.5,
        heart_rate=95.2,
    )

    time.sleep(60)
```

| code | 说明 |
|------|------|
| 0 | 成功 |
| 404 | 设备不存在 |
| 409 | 设备已注册 |
