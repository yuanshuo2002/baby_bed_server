#音色克隆小程序端对接指南

## 概述

本文档描述小程序端如何实现语音克隆功能，包括：克隆音色、选择音色、语音合成使用音色。

## 完整流程图

```
┌─────────────────────────────────────────────────────────────┐
│                    小程序端                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 用户点击"克隆音色"                                       │
│         ↓                                                    │
│  2. 选择音频文件（mp3/wav） │
```

## API 接口说明

### 1. 克隆音色（文件上传方式）

**接口：** `POST /api/v1/voice/library/upload`

**请求头：**
```
Authorization: Bearer {token}
Content-Type: multipart/form-data
```

**表单字段：**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| voice_role | string | 是 | 角色：mom/dad/nanny/other |
| voice_name | string | 是 | 音色名称 |
| text | string | 是 | 音频对应的文本内容 |
| is_default | boolean | 否 | 是否设为默认（默认false） |
| audio_file | file | 是 | 音频文件（支持mp3/wav格式） |

**响应：**
```json
{
    "code": 0,
    "message": "音色克隆成功",
    "data": {
        "id": 1,
        "family_id": 1,
        "voice_id": "speech:leijun:xxx:4928cf2f",
        "voice_role": "mom",
        "voice_name": "妈妈音色",
        "similarity_score": 0.85,
        "is_default": false,
        "created_at": "2026-06-10T12:00:00"
    }
}
```

**错误响应（不支持的音频格式）：**
```json
{
    "code": 500,
    "message": "Error opening <_io.BytesIO object at 0x7151b8ed3510>: Format not recognised."
}
```

---

### 2. 创建音色库记录（手动指定voice_id）

**接口：** `POST /api/v1/voice/library`

**说明：** 直接创建记录，不调用克隆API。用于手动管理音色。

**请求头：**
```
Authorization: Bearer {token}
Content-Type: application/json
```

**请求体：**
```json
{
    "voice_role": "mom",
    "voice_name": "妈妈音色",
    "voice_id": "speech:leijun:xxx:4928cf2f",
    "is_default": false
}
```

---

### 3. 获取音色库列表

**接口：** `GET /api/v1/voice/library`

**请求头：**
```
Authorization: Bearer {token}
```

**响应：**
```json
{
    "code": 0,
    "data": [
        {
            "id": 1,
            "voice_id": "speech:leijun:xxx:4928cf2f",
            "voice_role": "mom",
            "voice_name": "妈妈音色",
            "is_default": true
        },
        {
            "id": 2,
            "voice_id": "voice_002",
            "voice_role": "dad",
            "voice_name": "爸爸音色",
            "is_default": false
        }
    ]
}
```

**数据隔离逻辑：**
- 每个家庭只能看到自己的音色库
- 后端通过 JWT token 获取用户所属的 family_id
- 查询时自动过滤 `family_id = 当前家庭`
- 不需要小程序传 family_id，后端自动处理

**小程序调用示例：**
```javascript
async getVoiceLibrary() {
  const res = await wx.request({
    url: 'https://your-api.com/api/v1/voice/library',
    method: 'GET',
    header: { 'Authorization': 'Bearer ' + token }
  });
  if (res.data.code === 0) {
    this.setData({ voiceList: res.data.data });
  }
}
```

---

### 4. 切换音色

**接口：** `POST /api/v1/voice/switch`

**请求体：**
```json
{
    "voice_id": "speech:leijun:xxx:4928cf2f"
}
```

**响应：**
```json
{
    "code": 0,
    "message": "音色切换成功",
    "data": {
        "voice_id": "speech:leijun:xxx:4928cf2f",
        "voice_name": "妈妈音色",
        "voice_role": "mom",
        "is_default": true
    }
}
```

---

### 5. 根据音色名称删除音色

**接口：** `DELETE /api/v1/voice/library/by-name/{voice_name}`

**说明：** 根据音色名称删除音色，只删除当前家庭的音色，自动调用外部API删除对应voice_id

**请求头：**
```
Authorization: Bearer {token}
```

**路径参数：**
| 参数 | 说明 |
|------|------|
| voice_name | 音色名称（如"妈妈音色"） |

**响应：**
```json
{
    "code": 0,
    "message": "音色\"妈妈音色\"删除成功"
}
```

**错误响应（音色不存在或不属于当前家庭）：**
```json
{
    "code": 404,
    "message": "音色\"妈妈音色\"不存在或不属于当前家庭"
}
```

---

### 6. 语音合成（TTS）

**接口：** `POST /api/v1/voice/tts`

**说明：** 将文本转换为语音，自动调用外部 `generate_speech` 接口

**请求头：**
```
Authorization: Bearer {token}
Content-Type: multipart/form-data
```

**表单字段：**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| baby_id | int | 是 | 宝宝ID |
| text | string | 是 | 待合成的文本内容 |
| voice_id | string | 否 | 音色ID（不传则使用当前默认音色） |

**响应：**
```json
{
    "code": 0,
    "message": "语音合成成功",
    "data": {
        "baby_id": 1,
        "voice_id": "speech:leijun:xxx:4928cf2f",
        "voice_name": "妈妈音色",
        "audio_data": "524946466...",
        "text": "宝宝，该睡觉了"
    }
}
```

**后端自动调用外部API的参数：**
```json
{
    "input": "宝宝，该睡觉了",
    "voice": "speech:leijun:xxx:4928cf2f",
    "response_format": "wav",
    "sample_rate": 24000,
    "stream": false,
    "speed": 1.0
}
```

## 小程序端代码示例

### 页面 JS

```javascript
// pages/voice/voice.js
Page({
  data: {
    voiceList: [],
    currentVoice: null,
    inputText: '',
    selectedFile: null,
    selectedRole: null,
    inputText: '',  // 音频对应的文本
  },

  // ========== 1. 克隆音色 ==========

  // 1.1 用户点击"克隆音色"按钮
  onCloneVoice() {
    // 选择音频文件
    wx.chooseMessageFile({
      count: 1,
      type: 'file',
      extension: ['mp3', 'wav'],
      success: (res) => {
        const file = res.tempFiles[0];
        this.setData({ selectedFile: file });
        // 弹出角色选择
        this.showRolePicker();
      }
    });
  },

  // 1.2 弹出角色选择弹窗
  showRolePicker() {
    wx.showActionSheet({
      itemList: ['妈妈音色', '爸爸音色', '保姆音色', '其他音色'],
      success: (res) => {
        const roles = ['mom', 'dad', 'nanny', 'other'];
        const role = roles[res.tapIndex];
        this.setData({ selectedRole: role });
        // 弹出文本输入对话框
        this.showTextInputDialog();
      }
    });
  },

  // 1.3 弹出文本输入对话框
  showTextInputDialog() {
    wx.showModal({
      title: '请输入音频对应的文本',
      editable: true,
      placeholderText: '请朗读这段文字...',
      success: (res) => {
        if (res.confirm && res.content) {
          this.setData({ audioText: res.content });
          this.confirmClone();
        }
      }
    });
  },

  // 1.4 确认克隆（文件上传方式）
  async confirmClone() {
    const { selectedFile, selectedRole, audioText } = this.data;

    if (!selectedFile || !selectedRole || !audioText) {
      wx.showToast({ title: '请完成所有步骤', icon: 'none' });
      return;
    }

    wx.showLoading({ title: '克隆中，请稍候...' });

    try {
      const res = await wx.uploadFile({
        url: 'https://your-api.com/api/v1/voice/library/upload',
        filePath: selectedFile.path,
        name: 'audio_file',
        formData: {
          voice_role: selectedRole,
          voice_name: this.getRoleName(selectedRole),
          text: audioText,
          is_default: false,
        },
        header: {
          'Authorization': 'Bearer ' + getApp().globalData.token,
        }
      });

      wx.hideLoading();

      const data = JSON.parse(res.data);
      if (data.code === 0) {
        wx.showToast({ title: '克隆成功' });
        this.getVoiceLibrary();
      } else {
        wx.showToast({ title: data.message || '克隆失败', icon: 'none' });
      }
    } catch (err) {
      wx.hideLoading();
      wx.showToast({ title: '网络错误', icon: 'none' });
    }
  },

  getRoleName(role) {
    const names = { mom: '妈妈音色', dad: '爸爸音色', nanny: '保姆音色', other: '其他音色' };
    return names[role] || '自定义音色';
  },

  // ========== 2. 获取音色列表 ==========

  onLoad() {
    this.getVoiceLibrary();
  },

  async getVoiceLibrary() {
    try {
      const res = await wx.request({
        url: 'https://your-api.com/api/v1/voice/library',
        method: 'GET',
        header: { 'Authorization': 'Bearer ' + getApp().globalData.token }
      });

      if (res.data.code === 0) {
        this.setData({ voiceList: res.data.data });

        // 找出默认音色
        const defaultVoice = res.data.data.find(v => v.is_default);
        if (defaultVoice) {
          this.setData({ currentVoice: defaultVoice });
          wx.setStorageSync('current_voice_id', defaultVoice.voice_id);
          wx.setStorageSync('current_voice_name', defaultVoice.voice_name);
        }
      }
    } catch (err) {
      console.error('获取音色列表失败', err);
    }
  },

  // ========== 3. 选择音色 ==========

  async selectVoice(e) {
    const voice = e.currentTarget.dataset.voice;

    try {
      const res = await wx.request({
        url: 'https://your-api.com/api/v1/voice/switch',
        method: 'POST',
        header: {
          'Authorization': 'Bearer ' + getApp().globalData.token,
          'Content-Type': 'application/json'
        },
        data: {
          voice_id: voice.voice_id
        }
      });

      if (res.data.code === 0) {
        this.setData({ currentVoice: voice });
        wx.setStorageSync('current_voice_id', voice.voice_id);
        wx.setStorageSync('current_voice_name', voice.voice_name);
        wx.showToast({ title: '已切换为：' + voice.voice_name });
      } else {
        wx.showToast({ title: res.data.message || '切换失败', icon: 'none' });
      }
    } catch (err) {
      wx.showToast({ title: '网络错误', icon: 'none' });
    }
  },

  // ========== 4. 语音合成 ==========

  onInputText(e) {
    this.setData({ inputText: e.detail.value });
  },

  async onTextToSpeech() {
    const text = this.data.inputText;
    if (!text) {
      wx.showToast({ title: '请输入文字', icon: 'none' });
      return;
    }

    const voiceId = wx.getStorageSync('current_voice_id');
    // voiceId 可选，不传则使用默认音色

    wx.showLoading({ title: '合成中...' });

    try {
      const res = await wx.request({
        url: 'https://your-api.com/api/v1/voice/tts',
        method: 'POST',
        header: {
          'Authorization': 'Bearer ' + getApp().globalData.token,
        },
        data: {
          baby_id: this.data.babyId,
          text: text,
          voice_id: voiceId || ''
        }
      });

      wx.hideLoading();

      if (res.data.code === 0 && res.data.data.audio_data) {
        // 将十六进制音频数据转换为播放
        const arrayBuffer = this.hexToArrayBuffer(res.data.data.audio_data);
        const audio = wx.createInnerAudioContext();
        // 使用临时文件路径播放
        const filePath = `${wx.env.USER_DATA_PATH}/tts_audio.wav`;
        const fs = wx.getFileSystemManager();
        fs.writeFile({
          filePath: filePath,
          data: arrayBuffer,
          encoding: 'binary',
          success: () => {
            audio.src = filePath;
            audio.play();
            audio.onError(() => {
              wx.showToast({ title: '播放失败', icon: 'none' });
            });
          }
        });
      } else {
        wx.showToast({ title: res.data.message || '合成失败', icon: 'none' });
      }
    } catch (err) {
      wx.hideLoading();
      wx.showToast({ title: '网络错误', icon: 'none' });
    }
  },

  //十六进制字符串转 ArrayBuffer
  hexToArrayBuffer(hex) {
    const buffer = new ArrayBuffer(hex.length / 2);
    const view = new DataView(buffer);
    for (let i = 0; i < hex.length; i += 2) {
      view.setUint8(i / 2, parseInt(hex.substr(i, 2), 16));
    }
    return buffer;
  }
});
```

## 数据表结构

```sql
CREATE TABLE `family_voice_presets` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '主键ID',
  `family_id` bigint NOT NULL COMMENT '家庭ID',
  `voice_id` varchar(64) NOT NULL COMMENT '音色ID(外部API返回的uri)',
  `voice_role` enum('mom','dad','nanny','other') NOT NULL COMMENT '角色',
  `voice_name` varchar(100) DEFAULT NULL COMMENT '音色名称',
  `audio_url` varchar(500) DEFAULT NULL COMMENT '音频URL',
  `duration_ms` int DEFAULT NULL COMMENT '音频时长(ms)',
  `similarity_score` decimal(3,2) DEFAULT NULL COMMENT '相似度评分',
  `is_default` tinyint DEFAULT '0' COMMENT '是否默认音色',
  `is_active` tinyint DEFAULT '1' COMMENT '是否有效',
  `sort_order` int DEFAULT '0' COMMENT '排序顺序',
  `created_at` datetime DEFAULT CURRENT_TIMESTAMP,
  `updated_at` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_voice_id` (`voice_id`),
  KEY `idx_family_id` (`family_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='家庭音色库表';
```

## 注意事项

1. **音频格式**：支持 mp3、wav 格式
2. **音频大小**：建议不超过 10MB
3. **克隆时间**：外部 API 克隆可能需要 10-30 秒，请显示 loading
4. **文本输入**：必须提供音频对应的文本内容（text 参数）
5. **错误处理**：格式不支持时返回 "Format not recognised" 错误
6. **本地缓存**：切换音色后保存到 localStorage，下次打开自动使用
