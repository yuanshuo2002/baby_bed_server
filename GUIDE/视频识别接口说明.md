# 视频上传接口文档

## 概述

硬件端视频上传和管理相关接口。

## 数据库表

```sql
CREATE TABLE IF NOT EXISTS `videos` (
  `id` BIGINT NOT NULL AUTO_INCREMENT,
  `device_id` VARCHAR(50) NOT NULL,
  `video_url` VARCHAR(500) NOT NULL,
  `video_content_text` TEXT NULL COMMENT '视频AI理解后的文字内容',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
  `updated_at` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  INDEX `idx_device_id` (`device_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='视频文件信息表';
```

## 接口列表

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/video/upload` | 上传视频文件 |
| GET | `/api/v1/video/stream/{filename}` | 获取视频流 |
| POST | `/api/v1/video/add` | 插入视频识别结果 |
| GET | `/api/v1/video/get/{video_id}` | 根据ID查询 |
| PUT | `/api/v1/video/update/{video_id}` | 更新视频信息 |
| DELETE | `/api/v1/video/delete/{video_id}` | 删除视频 |
| GET | `/api/v1/video/by-device/{device_id}` | 根据设备ID查询 |
| GET | `/api/v1/video/list` | 分页查询 |

## 接口详情

### POST /api/v1/video/upload

上传视频文件到服务器。

**查询参数：**
- `device_id` (必填)：设备ID

**请求体：**
- `file`：视频文件（multipart/form-data）

**响应：**
```json
{
  "code": 0,
  "message": "视频上传成功",
  "data": {
    "id": 1,
    "video_url": "/api/v1/video/stream/abc123.mp4",
    "filename": "abc123.mp4"
  }
}
```

### GET /api/v1/video/stream/{filename}

获取视频流，支持直接播放。

**路径参数：**
- `filename`：视频文件名

**响应：**
- 视频文件流（video/mp4 等）

### POST /api/v1/video/add

插入视频识别结果。

**请求体：**
```json
{
  "device_id": "R60ABD1",
  "video_url": "https://example.com/video.mp4",
  "video_content_text": "宝宝正在睡觉，呼吸平稳"
}
```

**响应：**
```json
{
  "code": 0,
  "message": "视频信息插入成功",
  "data": {
    "id": 1
  }
}
```

### GET /api/v1/video/get/{video_id}

根据视频ID查询。

**响应：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "device_id": "R60ABD1",
    "video_url": "https://example.com/video.mp4",
    "video_content_text": "宝宝正在睡觉，呼吸平稳",
    "created_at": "2025-09-18T23:16:05",
    "updated_at": "2025-09-18T23:16:05"
  }
}
```

### PUT /api/v1/video/update/{video_id}

更新视频信息。

**请求体：**
```json
{
  "video_content_text": "更新后的视频内容描述"
}
```

### GET /api/v1/video/list

分页查询视频列表。

**查询参数：**
- `device_id` (必填)：设备ID
- `page_index` (可选，默认1)：页码
- `page_size` (可选，默认10)：每页数量

**响应：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "list": [...],
    "total": 100
  }
}
```
