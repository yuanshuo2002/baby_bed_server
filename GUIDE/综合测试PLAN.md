### 1.1 小程序端 (wechatapp)

- **技术栈**：UniApp + Vue 3 + TypeScript + Pinia + uView Plus
- **端口**：微信开发者工具
- **核心文件**：
  - `src/api/config.ts` - API配置 (119个接口)
  - `src/stores/` - 状态管理
  - `src/pages/` - 页面组件

### 1.2 硬件端 (baby_agent_system------2.0)

- **技术栈**：Python + 雷达API + 视觉API + 语音合成
- **运行环境**：Raspberry Pi / Windows
- **核心模块**：
  - `adapters/` - 各种API适配器
  - `agents/` - AI Agent (voice, video, notification, ui)
  - `config/` - 配置文件
  - `contracts/` - 命令和事件定义

### 1.3 后端API服务

- **地址**：`http://223.247.96.246:8123/api/v1`
- **接口数量**：119个
- **部署**：云端服务
