### 克隆音色 /library/upload

**完整URL：** `POST http://223.247.96.246:8123/api/v1/voice/library/upload`

**请求：** multipart/form-data
- voice_role: mom/dad/nanny/other
- voice_name: 音色名称
- text: 音频对应的文本
- is_default: 是否设为默认
- audio_file: 音频文件( mp3/wav)

**响应：**
```json
{
    "code": 0,
    "message": "音色克隆成功",
    "data": {
        "id": 1,
        "voice_id": "speech:leijun:xxx:de0059e2",
        "voice_name": "妈妈音色",
        "voice_role": "mom"
    }
}
```

### 根据音色名称删除 /library/by-name/{voice_name}

**完整URL：** `DELETE http://223.247.96.246:8123/api/v1/voice/library/by-name/{voice_name}`

**请求：**
```
DELETE http://223.247.96.246:8123/api/v1/voice/library/by-name/妈妈音色
Authorization: Bearer {token}
```

**响应：**
```json
{
    "code": 0,
    "message": "音色\"妈妈音色\"删除成功"
}
```

**处理流程：**
1. 根据 voice_name 查询本地音色库 (family_voice_presets)
2. 验证属于当前家庭的 family_id
3. 调用 DELETE http://223.247.96.246:30028/v1/audio/delete_voice?voice_id=xxx 删除外部音色
4. 软删除本地记录 (is_active=0)

### 语音合成 /tts

**完整URL：** `POST http://223.247.96.246:8123/api/v1/voice/tts`

**请求：** multipart/form-data
- baby_id: 宝宝ID
- text: 待合成文本
- voice_id: 音色ID（可选，不传则使用默认音色）

**响应：**
```json
{
    "code": 0,
    "message": "语音合成成功",
    "data": {
        "voice_id": "speech:leijun:xxx:de0059e2",
        "audio_data": "524946466...",
```

```
    "voice": "speech:leijun:xxx:de0059e2",
```
