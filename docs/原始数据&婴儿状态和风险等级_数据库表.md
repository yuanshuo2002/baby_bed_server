# 硬件端数据存储说明

## 概述

硬件端上传的数据分别存储在两个数据库表中：

| 接口 | 数据库表 | 说明 |
|------|---------|------|
| `/sensor/upload` | `sensor_data_raw` | 传感器原始数据 |
| `/sensor/status/upload` | `baby_status_log` | 婴儿状态及风险等级 |

---

## 1. sensor_data_raw（传感器原始数据）

记录设备采集的原始传感器数据。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键，自增 |
| device_sn | varchar(64) | 设备序列号 |
| baby_id | bigint | 宝宝ID |
| collected_at | datetime | 采集时间 |
| breath_rate | decimal(5,2) | 呼吸频率 |
| heart_rate | decimal(5,2) | 心率 |
| body_movement | decimal(5,2) | 体动活跃度 |
| distance_cm | decimal(6,2) | 雷达测距（cm） |
| sound_db | decimal(5,2) | 声音分贝 |
| sound_type | varchar(20) | 声音类型 |
| pose_status | varchar(20) | 姿态状态 |
| face_detected | int | 是否检测到面部（0/1） |
| expression | varchar(20) | 表情分类 |
| body_offset_cm | decimal(5,2) | 身体距床边距离（cm） |
| roll_angle | decimal(5,2) | 翻转角度 |
| height_cm | decimal(5,2) | 头部高度（cm） |
| room_temp | decimal(4,1) | 室温（℃） |
| humidity | decimal(4,1) | 湿度（%） |
| noise_db | decimal(5,2) | 环境噪声（dB） |
| created_at | datetime | 创建时间 |

---

## 2. baby_status_log（婴儿状态日志）

记录婴儿状态及风险等级。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键，自增 |
| baby_id | bigint | 宝宝ID |
| device_sn | varchar(64) | 设备序列号 |
| status_type | enum | 状态类型 |
| status_level | int | 状态级别 |
| started_at | datetime | 状态开始时间 |
| ended_at | datetime | 状态结束时间（可空） |
| duration_sec | int | 持续时长（秒） |
| breath_rate | decimal(5,2) | 触发时呼吸频率（快照） |
| heart_rate | decimal(5,2) | 触发时心率（快照） |
| sound_db | decimal(5,2) | 触发时声音分贝（快照） |
| pose_status | varchar(20) | 触发时姿态（快照） |
| expression | varchar(20) | 触发时表情（快照） |
| screen_triggered | int | 是否触发屏幕响应 |
| sound_triggered | int | 是否触发声音响应 |
| push_triggered | int | 是否推送通知 |
| note | text | 备注 |
| created_at | datetime | 创建时间 |

### status_type 可选值

| 值 | 说明 |
|----|------|
| sleeping | 熟睡 |
| awake | 刚醒 |
| playing | 高兴玩耍 |
| crying | 哭闹 |
| danger | 危险动作 |

### status_level 可选值

| 值 | 说明 |
|----|------|
| 0 | 正常 |
| 1 | 轻微关注 |
| 2 | 警告 |
| 3 | 危险 |

---

## 数据关系

```
device (device_sn)
    │
    ├──► sensor_data_raw（多条记录，原始传感器数据）
    │         └── 每个采集周期一条记录
    │
    └──► baby_status_log（多条记录，状态变化）
              └── 每次状态变化一条记录
              └── 同一时刻只有一条 ended_at 为空的记录
```
